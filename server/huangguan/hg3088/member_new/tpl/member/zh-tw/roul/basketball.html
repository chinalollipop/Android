籃球

  • NBA/NBL籃球賽事開始後至少要進行43分鐘才被視為有效，其他的籃球聯賽，至少要超過35分鐘才被視為有效，除非其它賽事特殊規定。
  • 投注特定時間段的賽事(如單節投注或半場投注)，若全場賽事未完成，但相關時段的賽事已結束，投注在該時段內的注單均被視為有效。
  • 上半場的賽果包括第一節和第二節的分數總和。下半場的賽果包括第三節和第四節的分數總和。
  • 所有投注在全場和下半場的都包括加時得分。
  • 所有投注的贏/輸以全程比分為準(包括加時得分)，提供的賽事如以上所說至少進行43分鐘（35分鐘）或另有說明。如果賽事在43分鐘以後終止，同時沒有在官方開賽時間12小時內重新開始，最終的比分會以賽事終止前的比分為準，無論賽事是否在之後完成。為了避免疑慮，如果賽事在官方開賽12小時內重新開始，此類賽事的結果將不被考慮。
  • 投注在第四節是不包括加時得分。
  • 美國大學生籃球聯賽- 全場時間為40分鐘（每半場時間為20分鐘）。只有當賽事進行35分鐘或以上，所有的投注才會被視為有效。
  • 美國大學生籃球聯賽- 溫馨提示，網站頁面的主場和客場僅供參考。無論是否更改主場，客場或中立場，所有註單都被視為有效。如果賽事在官方開賽時間提前開始，所有註單投注在開賽前都被視為有效，所有註單投注在開賽后都被視為無效。

  • 投注種類

  • 上半場籃球滾球投注會在球賽開始後開出，球賽的比分會以0—0開始。
  • 如果賽事在上半場終止，而賽事在官方開賽時間12小時內重新開始，所有上半場滾球投注被視為有效。
  • 如果賽事在上半場終止，而賽事在官方開賽時間12小時後重新開始，所有上半場滾球投注被視為無效，返回本金。
  • 如果賽事在上半場結束後終止，所有投注在上半場籃球滾球的注單被視為有效。

  • 單式（美式盤口）

  • 是當場賽事比分的盤口，不是主場就是客場。

  • 讓分盤
  • 賠率前顯示讓球數的球隊表示此隊是讓球方，而另一對是受讓方。在比賽正式開始前，受讓方已得到讓球方讓的虛擬分數， 以領先的情況下開始比賽。
  • 總分數

  • 此盤口類型是預測比賽最終的總比分。

  • 單/雙
  • 此盤口類型是以全場賽事的總比分競猜單/雙，賽果0-0被視為雙數。
  • 上半場籃球滾球
  • 上半場籃球滾球投注會在賽事開始後開出，球賽的比分會以0—0開始。
  • 如果賽事在上半場終止，而賽事在官方開賽時間12小時內重新開始，所有上半場滾球投注被視為有效。
  • 如果賽事在上半場終止，而賽事在官方開賽時間12小時後重新開始，所有上半場滾球投注被視為無效，返回本金。
  • 如果賽事在上半場已結束後終止，所有上半場滾球投注被視為有效。
  • 下半場籃球滾球
  • 下半場籃球滾球投注會在下半場賽事開始後開出，球賽的比分會以0—0開始。
  • 如果賽事在下半場終止，而賽事在官方開賽時間12小時內重新開始，所有下半場滾球投注被視為有效。
  • 如果賽事在下半場終止，而賽事在官方開賽時間12小時後重新開始，所有下半場滾球投注被視為無效，返回本金。
  • 下半場籃球滾球投注包括加時/補時。
  • 全場籃球滾球
  • 球賽比分將以0—0開始計算。
  • 如果賽事終止，而賽事在官方開賽時間12小時內重新開始，所有投注全場籃球滾球的注單被視為有效。
  • 如果賽事終止，而賽事在官方開賽時間12小時後重新開始，所有投注全場籃球滾球的注單被是為無效， 返回本金。
  • 全場籃球滾球投注包括加時/補時。

滾球讓分盤和大小盤

  • 投注籃球滾球讓球和大小盤口類型，出於結算的目的，賽​​事比分從0—0開始，無論投注時的實際比分是多少。

    範例:

    主/客 讓分盤 大小盤
    邁阿密熱火 3.5 0.90 90.5 0.90
    休斯頓火箭 0.90 0.90
  • 根據以上盤口顯示，可以清楚看到，如果投注（邁阿密熱火）獲勝，由於邁阿密熱火讓休斯頓火箭：3.5分，因此最後結果邁阿密熱火至少需贏休斯頓火箭四分或以上方可獲勝。而由於大小球所開設的盤口是大小球90.5，因此結果可以看作，賽事結束後，如果兩隊的總分數等於或大於91分，投注大則為贏，投注小則為輸。反之，如果兩隊的總分數等於或小於90，投注為小則為贏，投注大則為輸。

首個/最後得分的球隊

  • 預測首個或最後一個得分的球隊。
  • 開賽后一旦球隊得分，所有投注在首個球隊得分的注單被視為有效。
  • 如果賽事終止，而賽事沒有在官方開賽時間內12小時內重新開始，所有投注在最後一個得分球隊的注單被視為無效。

籃球特別投注

  • 預測個人得分，搶斷，籃板，助攻，三分球等。
  • 賽果將以官方賽事結果為準，包括加時。
  • 雙方球員/球隊必需參賽投注才被視為有效。
  • 如果一方或雙方球員/球隊沒有參與比賽，所有投注被視為無效，返回本金。
  • 注單的輸贏將根據NBA或FIBA官方所公佈的賽果為準，賽后的任何數據更改被視為無效

夢幻籃球規則

  • 夢幻籃球投注的遊戲方法是從同一天的任何兩場賽事中（沒有關聯）選出兩支球隊投注。
  • 夢幻籃球賽投注結果將依據所選的球隊在實際球賽中的進球分數來計算，如有讓球，則將依讓球後得出的結果為準。
  • 兩支球隊必需在同一天比賽投注才被視為有效。
  • 如果所選的球隊並非在公司網站顯示時間進行比賽，所有與該球隊有關的夢幻賽事投注被視為無效。
  • 夢幻籃球賽事中，球賽進行的場地不在遊戲的考慮範圍內。

    例如 ；籃球賽事在同一天進行。
    波士頓凱爾特人101:98 芝加哥公牛，洛杉磯湖人118:101 奧蘭多魔術
    夢幻賽事1，波士頓凱爾特人VS 洛杉磯湖人
    波士頓凱爾特人 101
    洛杉磯湖人 118
    賽果將根據各自賽事實際的結果為準。

  • 所選的球隊一定要完成實際的球賽，球賽的比分需經過球賽官方鑑定，那麼與該球隊有關的夢幻賽投注才被視為有效。如果所選球隊沒有完成比賽或官方否定最終結果，那麼與該球隊有關的夢幻賽投注被視為無效，返回本金到會員賬號。
  • 所有的夢幻賽都會按照現有的籃球規則和標準執行。

單節最高得分球隊

  • 投注是指每個賽事中選擇得分最高的單節比分結算，不包括加時賽。

最先獲得20分的球隊。

  • 投注是指每個賽事中每節最先獲得20分的球隊。
  • 滾球：盤口將會根據情況進行調整，例如10，15，20分的盤口類型。

單節籃球滾球投注

  • 單節需要完全結束注單才被是為有效，除非賽果已經確認，不包括加時

最高單節得分

  • 所有四節都需要完成，不包括加時。
  • 如果賽事終止，投注被是為無效，除非賽事在第四節終止時，其中一個隊伍獲得最高分數。
