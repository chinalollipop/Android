選擇體育項目 :

足球

  • 所有足球賽事的投注皆以球賽所規定的完場時間2x45分鐘為準（某特定賽事或友誼賽為2x40分鐘），包括球員受傷的補時。至於加時賽、淘汰賽、點球，以及賽后如裁判或官方更改任何賽果則不計算在內。
  • 有少年賽、友誼賽的完場時間為2x35分鐘或更少，皇冠投注網將在賽前作出公佈，否則該場賽事的投註一律作廢。

回最上層
  • 獨贏
  • 於指定球賽中，兩隊以平手的方式(不計算讓球在內)，選擇其中一隊勝出或和局，賽果以90分鐘之法定時間球賽結果為準。
  • 上半場滾球獨贏盤
  • 投注皆以球賽上半場的完場時間45分鐘為準（某特定賽事或友誼賽為40分鐘），包括球員受傷的補時。
  • 輸贏將會根據上半場的完場比分而定。
  • 範例::
  • 聯賽 時間   目前的比分 滾球獨贏盤
    英格蘭超級聯賽 21/07 1:00PM 阿仙奴(主) 1 1.5
    曼聯(客) 0 4.5
    和   2.8
  • 在賽事比分為1-0時，投注上半場滾球獨贏盤- 阿仙奴
    如阿仙奴VS曼聯的上半場完場比分停留在1-0
    所有投注在上半場滾球獨贏盤- 阿仙奴(主場)的注單視為贏
    （ 投注在客場或平手的注單將視為輸，主場贏）
  • 在賽事比分為1-0時，投注上半場滾球獨贏盤- 曼聯
    如阿仙奴VS曼聯的上半場完場比分停留在1-0
    所有投注在上半場滾球獨贏盤- 曼聯(客場)的注單視為輸
    （ 投注在客場或平手的注單將視為輸，主場贏）
回最上層
  • 滾球獨贏盤
  • 除非特別註明，所有投註一律以90分鐘的球賽賽果為準，不包括加時得分。
  • 加時賽將視為一場新的賽事，公司將另外開出加時賽盤口。所有的投注都將​​依加時賽的結果進行結算。
  • 滾球獨贏盤的輸贏將會根據全場的完場比分而定。賽果以90分鐘之法定時間球賽結果為準，不包括加時得分。
  • 範例:
  • 聯賽 時間   目前的比分 滾球獨贏盤
    英格蘭超級聯賽 21/07 1:00PM 阿仙奴(主) 1 1.5
    曼聯(客) 0 4.5
    和   2.8
  • 在賽事比分為1-0時，投注滾球獨贏盤- 阿仙奴
    如阿仙奴VS曼聯全場的完場比分是2-1
    所有在賽事比分為1-0時投注阿仙奴的注單視為贏
    （ 投注在客場或平手的注單將視為輸，主場贏）
  • 在賽事比分為1-0時，投注滾球獨贏盤- 阿仙奴
    如阿仙奴VS曼聯的完場比分是1-1
    所有投注在滾球獨贏盤- 曼聯(客場)和阿先奴（主場）的注單視為輸
    （ 投注在客場和主場的注單將視為輸，平手贏）
  • 讓球
  • 讓球盤----即在比賽正式開始前， 一方球隊已得到另一方讓的虛擬分數， 以領先的情況下開始比賽。
  • 如果您下注被讓球一方，比分需要在比賽最終比分上再加上虛擬領先分數，以此結果為準計算輸贏。
  • 讓球盤球賽分以下兩種讓球方式:
    • 在一場球賽中，其中一隊球隊開出一個讓球球盤，如讓"一球"（投注的金額被投注在讓"一球"的球盤上）
    • 在一場球賽中，其中一隊球隊開出二個讓球球盤，如讓"半球/一球"（投注的金額將被平分為兩份，一半投注在讓"半球"的球盤上，而另一半投注則投注在讓"一球"的球盤上）
  • 大/小盤
  • 大/ 小盤(總入球數): 由比賽最後的總比分決定，於指定球賽中當投注該場比賽的大/ 小盤時，若總入球數高於該場比賽的大/小盤球數則為大盤，若總入球數低於該場比賽的大/小盤球數則為小盤。
  • 大小盤球賽分以下兩種讓球方式:
    • 在一場球賽中，如大小盤"3球"（投注的金額被投注在"3球"大/小的球盤上）
    • 在一場球賽中，如開出二個大小盤"2.5球/3球"（投注的金額將被平分為兩份，一半投注在大/小"2.5球"的球盤上，而另一半投注則投注在大/小"3球"的球盤上）
  • 上半場---讓球盤/大小盤/獨贏
  • 投注上半場所有種類註單，均以上半場球賽結果為準。
  • 如上半場球賽取消或賽事被中斷，投注上半場所有註單一律取消。
  • 如下半場球賽取消或賽事被中斷，投注上半場所有註單均屬有效，不得取消。
回最上層
  • 單/雙
  • 預測一場比賽中的進球總數是單數還是雙數的投注方法，如果賽果0-0被視為雙數。
  • 烏龍球將被予以計算在內。
  • 標準過關
  • 在標準過關賽事中，可以選擇多個賽事串在一起一同下注。
  • 賽果以90分鐘之法定時間球賽結果為準。如果其中有場賽事沒有在法定時間進行，此場比賽將被視為無效，過關仍然有效，唯賠率以[1]計算。
  • 一張註單不可以下注重複的隊伍或選手。
  • 過關派彩的計算方式是用本金乘以其餘每場比賽的賠率。

  • EXAMPLE:

    聯賽 時間   主隊 和局 客隊  
    英格蘭超級聯賽 21/07
    1:00PM    		 阿仙奴(主) 1.70 3.20 4.50 曼聯
    英格蘭超級聯賽 21/07
    2:00PM    		 車路士(主) 1.70 3.30 4.40 富咸
    英格蘭超級聯賽 21/07
    3:00PM    		 利物浦(主) 1.30 4.00 8.50 熱刺

    投注範例1:
    投注: 阿仙奴VS 曼聯和局；切爾西贏；利物浦贏； 3串1，注金$1000，其結果為贏。 x 3.2 x 1.7 x 1.3 = 派彩: 1000x3.2x1.7x1.3=6072(派彩已經扣除本金)

    投注範例2:
    投注: 曼聯；切爾西；熱刺3串1，注金$1000，其結果為輸本金$1000

  • 讓球過關
  • 在讓球過關賽事中，可以選擇多個賽事串在一起一同下注。
  • 讓球過關的每一場球賽中，其中一隊球隊提供讓球作為比賽之條件。
  • 賽果以90分鐘之法定時間球賽結果為準。如果其中有場賽事沒有在法定時間進行，此場比賽將被視為無效。賠率以[1]計算。
  • 一張註單不可以下注重複的隊伍或選手
  • 讓球過關中，如果是輸半按乘以0.5計算，如果是贏半按（1+賠率/2）計算，如果是打平按（1+賠率）*0.5計算。如是是賽事取消或中斷，賠率按[1]計算。

回最上層
  • 總入球數
  • 所有的投註一律以該球隊在賽事中的最後總入球數進行結算。
  • 烏龍球將被予以計算在內。
  • 單一球隊總入球數
  • 所有的投註一律以該球隊在賽事中的最後總入球數進行結算。
  • 若球隊的最後總入球數超出盤口的總入球數為"大";低於盤口的總入球數為"小"。
  • 半/全場
  • 半/全場是指預測上半場及全場賽果的投注
  • 烏龍球將被予以計算在內。
  • 如果比賽場地更換，所有註單無效，將退還本金。
  • 首位/最後進球員
  • 在90分鐘法定的時間內，按照官方列出的出場名單中，預測首名或最後一名入球的球員。
  • 烏龍球不計為第一球或最後一球，如果出現烏龍球，下一個或之前的進球將被考慮有效。
  • 如投注的首位進球員並無參與該場賽事，或該球員在第一個進球後才進場，注單將被判無效。如在90分鐘的法定時間內唯一的進球是烏龍球，"其他"將是此場賽事的進球員。
  • 如果您選擇的最先進球的球員，在沒有射入第一個球就被罰下場或被其他球員替代，您的投注視為輸。
  • 最後進球者將包括任何參賽並且上場的球員。
  • 如果在射入第一個球以後球賽被取消，所有投注首名進球者的注單有效,而最後進球球員的注單將被取消。
  • 如果在沒有進一個進球之前，球賽就被取消，所有的投注在第一個進球的注單將被取消。
  • 所有的投注都將​​依90分鐘法定時間為準，不包括加時得分。
  • 進球球員
  • 從提供的球員列表中，選擇並預測在某特定比賽90分鐘內，該球員是否會有進球，
  • 如該球員未上場參賽，則該注單視為取消，返回本金。
  • 如球員在常規比賽任何時間登場，則注單有效。
  • 烏龍球與點球大戰不計算在內。
  • 球員進球數
  • 預測全場90分鐘內進球最多的球員，不包括加時賽以及點球賽。
  • 兩個球員都必須是比賽的先發球員，注單才有效。
  • 盤口僅限於雙方進行比賽的球員，按照顯示的盤口日期而定。
  • 烏龍球不被計算在內。
  • 單註投注規則：
    • 獨贏1x2：預測哪位球員進球數最多。如果兩位球員進球數一致，選擇平局結果的為勝出。包括兩位球員均無進球的情況。
    • 讓球：預測某球員的進球數最多，包括讓球分數。
  • 大/小球：預測兩位球員的總進球數是否大於或小於盤口球數
回最上層
  • 首名或最後進球的球隊
  • 在90分鐘法定的時間內，預測首名或最後進球的球隊。
  • 烏龍球將被視為有效。但烏龍球視為對方入球。比如: A隊VS B隊，B 隊踢進一個烏龍球，此球計為A隊獲得。
  • 投注在首名進球的球隊一旦賽果確定,不論賽事是否順利完場, 所有的注單都被視為有效。
  • 投注在最後進球的球隊如賽事無法順利完成,最後進球的球隊將視為無效。
  • 第一/最後一個角球
  • 預測在90分鐘的法定時間內，首先或最後一個獲得角球的球隊。
  • 若賽事在第一個角球出現後被取消,所有投注在"第一個角球"的注單都將被視為有效。
  • 若賽事在第一個角球出現後被取消,所有投注在"最後一個角球"的注單都將被視為無效。
  • 若賽事在第一個角球出現前被取消,所有投注在"第一/最後一個角球"的注單都將被視為無效。
  • 若賽事在所規定的完場時間90分鐘內並無出現任何角球，所有投注在" 第一/最後一個角球"的注單都將被取消。
  • 總角球數
  • 獲勝交易必須準確猜中角球數（例如: 8個角球），或者準確猜中角球數的範圍（例如: 14-16個角球）。被裁定但並未實際執行的角球﹐不予計算。若賽事在90分鐘之前會終止，交易作廢，如果角球被重新進行（例如: 在禁區內犯規），在此我們計算為一個角球。
  • 根據官方賽果或權威機構判定的結果為準。
  • 總角球數類似於亞洲盤，輸贏是由兩隊的總入角球數決定。
  • 所有的投注都將​​依90分鐘法定時間為準，不包括加時得分。
  • 如果某角球被重新進行(例如，在禁區內犯規)，仍計為一個角球。
  • 特定聯賽中主客場的進球數
  • 若聯賽中有任何一場賽事被中斷或取消，所有投注在"特定聯賽中的主客場進球數"的注單都將視為無效。
  • 在特定聯賽中的主客場進球數盤口與讓分盤或大小盤大致相同。贏/輸是根據主場與客場的最終總得分，再加上虛擬領先分數而定，以此結果為準。
  • 先開球球隊
  • 預測首先開球的球隊。
  • 比賽一旦開始,不論賽事是否順利完場, 所有的注單都被視為有效。
  • 夢幻對壘賽遊戲規則
  • 夢幻對壘賽的遊戲方法是從在同一天進行的任何2場(沒有關聯)球賽中選出2支球隊。
  • 球賽進行的地點不在遊戲的考慮範圍內。

    舉​​例說明: 在同一天舉行的球賽有

    阿仙奴0:0 曼聯、切爾西2:1 富咸、 利物浦3:1 熱刺,、曼市2:1 阿士東維拉

    夢幻對壘賽（一）、阿仙奴v 富咸
    依據所選的球隊在實際球賽中的進球來計算，夢幻對壘賽的比數是: 阿仙奴0，富咸1，賽果為0-1

    夢幻對壘賽（二）、切爾西v 熱刺
    依據所選的球隊在實際球賽中的進球來計算，夢幻對壘賽的比數是: 切爾西2熱刺1，賽果為2-1


  • 兩支所選的球隊的實際球賽必須是在同一天舉行，夢幻對壘賽的投注才算有效。
  • 如果所選的球隊的實際球賽賽期和我們網站顯示的賽期不同，與該球隊有關的夢幻對壘賽將視為無效。
  • 所選的球隊一定要完成實際的球賽，其球賽的比數成績須經過球賽主辦當局（如英格蘭超級足球聯賽）的鑑定，與該球隊有關的夢幻對壘賽的投注才算有效。
  • 所有夢幻對壘賽將遵守所有現存的足球規則和條款。
回最上層
  • 入球最多的球員
  • 如果有多於1個球員達到最高入球，結算將根據所有入球最高的球員平分。
  • 所有參加比賽的選手均將視為"參賽球員"。

    • 角球
    • 12碼罰球
    • 自由球(攻方在守方禁區附近的自由球)
    • 擲入球(攻方靠近守方禁區的擲入球)
    • A隊向B隊禁區附近進攻(或是B隊在A隊的禁區附近進攻)
    • 在以下的狀況，會員所下註註單將會被取消，而會員所下注的金額將會全數退回
    • 進球
    • 紅牌
    • 在以下這幾種狀況將會釋放危險球，而會員在危險球下注的注單都會以當時下注的盤口、賠率和金額為準。
    • 門將開球
    • 皮球重新去到中場線
    • 己方球員在自己的後場搶回控球權
    • 攻防轉換，原來的防守一方在自己後場開出自由球
    • 任何不確定的因素發生，都會影響特別投注的最終結果。
