連串過關/複式過關/組合過關

示意

連串過關是指選擇二個或更多的賽事在一個單一的注單中。每一個選擇連串投注的賽事必須獲勝此連串的注單才視為獲勝。如果第一個注單是獲勝的，投注獲勝的注單會添加到第二個投注選項，直到連串過關中的所有投注獲勝或到有一場失敗為結束。

例如：

投註一個100元的3串1連串過關在以下的三場賽事:

  • Manchester United @ 1.80曼聯
  • Chelsea @ 1.50 切爾西
  • Arsenal @ 1.66 阿森納

如果所有的三場賽事都獲勝，連串過關的盈利為448.20元（包括本金）。詳細的計算方式您可以查看以下的信息:

  • Bet 1: Manchester United 1.80 x $100 = Return of $180.曼聯1.8*100=180
  • Bet 2: Chelsea 1.50 x $180 = Return of $270.切爾西1.5*180=270
  • Bet 3: Arsenal 1.66 x $270 = Return of $448.20.阿森納1.66*270=448.20

Parlay Points:

  • 皇冠一個注單中最多的連串過關為10串。
  • 所有投注賽事都需要根據體育博彩規則為準。
  • 不是所有的賽事都可以投注連串過關。如果您在投注列表中看到不可以組合二個不相關的賽事（可以查看以下的信息關於有關聯的連串），那麼就是其中一場並沒有開出連串過關投注。
  • 連串過關可以投注不同種類的賽事，以及不在同一天的比賽。

RELATED SELECTION IN A PARLAYS

連串過關投注，選擇組合有關聯的同一賽事或投注市場的結果會影響其他另一個投注市場，此註單是不接受的。例如:

例如:

以下的2串1是不接受的，由於是同一場賽事:

  • 曼聯獲勝獨贏盤口@1.80
  • 曼聯2-0獲勝，正確比分盤口@7.0

如果曼聯2-0獲勝，組合盤口為12.6。其實盤口應該為7.0，因為曼聯2-0獲勝，那麼獨贏盤口自然而然為獲勝。

連串過關投注，選擇組合有關聯的同一球隊或球員，即使他們是不同的時間，同樣是不接受例如一個結果會影響另一個結果。

例如:

  • 曼聯進入到最後的冠軍杯決賽@6.0
  • 曼聯進入到最後的冠軍杯決賽@6.0
  • 組合盤口@60.0

這個連串過關被視為第二個賽果會影響到第一個賽果。如果曼聯獲得冠軍杯聯賽冠軍，那麼曼聯自然而然就進入冠軍杯決賽。因此，盤口僅僅為10.0。

皇冠有權利取消有關聯的連串過關投注。

連串過關

在連串過關中有任何的投注賽事無效或者打和（如以下的範例），此連串過關注單仍然有效，並且按照任何所勝出的其餘投注結算，範例如下：

  • 投注項1：切爾西（-0.5）-切爾西贏。
  • 投注項2:投注2：曼聯（-1）-曼聯贏1-0.
  • 投注項3:投注3: 阿森納（-0.5）-阿森納贏。

正如曼聯是以（-1）的亞洲盤口1-0獲勝，但在連串過關中的此賽事視為和。因此，當切爾西獲勝連串阿森納獲勝過關，此連串過關將被視為切爾西以及阿森納的2串，而非最初的3串。同時，打和的過關投注項目將會被自動以1計算。

過關的計算範例如下

範例1： 其中一個投注項為和:

投注項 讓球 赔率 賽過 結果
切爾西 (-0.5/1) 1.85 贏 2-0 全贏
曼聯 (-1) 1.95 贏 1-0 和
阿森納 (-1/1.5) 2.05 贏3-0 全贏

投注金額：$100 3串一

計算方式如下

$100 x 1.85 x 1 x 2.05 = $379.25,扣除本金$100=盈利$279.25

  • 切爾西(-0.5/1)：贏- $100 x 1.85 =返還$185。此金額將會移至下個選項。
  • 曼聯(-1): 和- $185 x 1 = $185. 此金額將會移至到下一選項。
  • 阿森納(-1/1.5)：贏- $185 x 2.05 =返還$379.25
  • 盈利：$379.25 – $100 = $279.25.

範例2： 其中一個投注項為贏半平半

投注項 讓球 赔率 賽過 結果
投注項 (-0.5/1) 1.85 贏 1-0 贏半 / 平半
曼聯 (-1) 1.95 贏 2-0 全贏
阿森納 (-1/1.5) 2.05 贏3-0 全贏

連串投注：$100 3串一

計算方式如下：

$100 x [1 + 0.5 x 0.85] x 1.95 x 2.05 = $569.64， 扣除本金=贏$469.64

  • 切爾西(-0.5/1)：贏半/平半– 此註單被分為兩項，只有一半的投注盈利，
    盈利的部分$50 x 1.85 = $92.50
    打和部分$50 x 1 = 50
    返還是$92.50 + $50 = $142.50。此金額將移至下個投注項
  • 曼聯（-1）:盈利-142.50 x 1.95=$277.87，此金額將移至下個投注項
  • 阿森納(-1/-1.5):盈利-277.87 x 2.05=$569.64
  • 總盈利:$569.64-$100=$469.64.

範例3：其中一個投注項為贏半平半

投注項 讓球 赔率 賽過 結果
切爾西 (-0.5/1) 1.85 贏2-0 贏
曼聯 (-1) 1.95 贏2-0 贏
阿森納 (-1/1.5) 2.05 贏 1-0 輸半 /平半

連串投注：$100 3串一

計算方式如下：

$100 x 1.85 x 1.95 x 0.5 = $180.38, 扣除本金$100 = 盈利$80.38

  • 切爾西(-0.5/1): 贏 - $100 x 1.85 = $185 = 返回of $185. 此金額將移至下個投注項
  • 曼聯(-1): 贏 - $185 x 1.95 = $360.75. 此金額將移至下個投注項
  • 阿森納(-1/1.5): 輸半/平半 – 投注額度將一分為二，其中一半為輸。
    $360.75 x 0.5 = $180.38.
    輸半: $180.38
    平半: $180.38 x 1 = $180.38
  • 返回: $180.38
  • 總盈利: $180.38 – $100 = $80.38.
