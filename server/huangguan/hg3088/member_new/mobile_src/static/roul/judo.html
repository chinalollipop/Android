柔道、摔交、跆拳道

最后更新日期: 04/03/2015

一般规则

  • 选手在颁奖仪式上所授予的金牌，银牌及铜牌分别认定为第一名，第二名及第三名。
  • 每场赛事的结果均以奖台上排名为最终结果。如之后发生丧失资格或结果修正将不予以计算。
  • 在投注二选一赛事时，只有双方参赛者均参与比赛时，注单才被为有效。
  • 如果比赛提前开赛，只有在开赛之前投注的注单被视为有效投注。在开赛之后投注的注单将被视为无效投注，滚球投注类型不包括在内。
  • 如果该场赛事没有举行颁奖仪式，所有柔道投注的输赢将依据 I.J.F（国际体联）公布的正式赛果而定。
  • 如果该场赛事没有举行颁奖仪式，所有摔交投注的输赢将依据 F.​I.L.A（国际体联）公布的正式赛果而定。
  • 如果该场赛事没有举行颁奖仪式，所有跆拳道投注的输赢将依据 W.T.F（国际体联）公布的正式赛果而定。
