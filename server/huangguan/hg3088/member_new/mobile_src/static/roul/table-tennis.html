一般规则

  • 所有注单必须以赛事结束方可视为有效，除非另有特别说明。
  • 如果比赛预定时间缩短，或者赛事获胜要求比分提高，所有注单一律无效并取消。
  • 如果赛事延期或中断，只要比赛最后有完成，所有投注将保持有效。
  • 如果比赛提前开赛，只有在开赛之前投注的注单将为视为有效投注。在开赛后投注的注单将被视为无效，滚球玩法除外。

投注类型

    独赢

    • 预测哪个球员将会赢取比赛的胜利。这个比赛包含双方球员。

    让盘 / 让局

    • 预测哪一位球员在盘口指定的让盘/局数赢得比赛。

    总盘/总局数： 大 / 小

    • 预测比赛进行的总盘/局数是大于或小于盘口指定的大小盘局数。

    让分

    • 预测在某个时节 哪一位球员在盘口指定的让分数赢得比赛。

    总分： 大 / 小

    • 预测提名的球员在某一特定的时间段内的个人总得分将大于或小于盘口指定的总分数线。

    球员/球队得分 -大/小

    • 预测主场或客场的总比分是大于还是小于指定盘口。
    • 如果赛事中断，除非市场已经派彩或任何将来的得分不会影响到赛果的投注有效，其他所有注单一律视为无效。

    波胆

    • 预测一个特定赛事全场的正确比分。
    • 结算将依据相关部门最终公布的比分为准。
    • 如果注单在赛事中断前已得到明确的胜负，并且任何进一步的赛果均不会对注单结果产生影响的情况下，注单会被视为有效。

    单/双

    • 预测在某一特定的时间段内比赛进行的总局数、盘数或分数是单数还是双数。
