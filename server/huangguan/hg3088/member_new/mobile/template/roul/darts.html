飞镖

最后更新日期: 08/12/2017

一般规则

  • 如果投注的选手没有参与比赛，所有所有投注此选手的注意将被取消。
  • 如果一个选手在比赛未结束前退出或被取消比赛资格，这场比赛的所有投注将视为无效，除非另有说明。
  • 如果一个选手并未在比赛结束前退出或被取消比赛资格，所有投注在此赛事上的注单将视为无效，除非另有说明。
  • 如果比赛在法定时间提前进行，只有在比赛开始前的投注仍然有效。比赛开始后的所有投注将被视为无效。（滚球投注另作别论）。

投注类型

    优胜者

    • 预测哪一位选手将会在指定比赛中胜出。
    • 如果比赛没有完成，相关机构所宣布的胜者玩家将会被视为优胜者。
    • 这个市场根据赛事可提供二项或者三项选择。
    • 如果二项投注中出现平局,那么所有注单将视为无效。
      • 平局的选项将包含在三项投注的选项中。

    优胜者（盘）

    • 预测哪一位选手将成为市场名单中指定盘的获胜者。
    • 如果指定盘未能完成，所有投注在该市场的注单将被视为无效。
    • 如果指定盘已完成，但是赛事没有完成，所有投注在这个市场的注单将被视为有效。

    (局) 优胜者

    • 预测哪一位选手将是市场名单中指定局的获胜者。
    • 局可能是特定的赛事其中一局的局数或特定赛事其中一盘的局数。
    • 如果指定局未能完成，所有在这个市场的投注将被视为无效。
    • 如果指定盘已完成，但是赛事没有完成，所有投注在这个市场的注单将被视为有效。

    波胆

    • 在比赛结束时预测正确的比分。
    • 如果比赛是基于“局”的话，每位选手的总局数记录将被采用。
    • 如果比赛是基于“盘“的话，每位选手的总盘数记录将被采用。
    • 如果全场比赛没有完成，所有的投注将会被视为无效。

    让球

    • 预测赛事在盘口指定的让球数上的结果。
    • 如果赛事是以“局”为基础，这个让球的提供将以最后局的得分为基础。
    • 如果赛事是以“盘”为基础，这个让球的提供将以最后盘的得分为基础
    • 如果比赛没有完成，那么所有的赌注将是无效的，除非赛事已有明确结果。

    总局数：大/小

    • 预测赛事的总局数将大于或小于赛事指定的大/小局分数。
    • 如果比赛没有完成，那么所有的赌注将是无效的，除非赛事已有明确结果。

    总局数-X盘

    • 预测在特定盘的范围内的确切的局数。
    • 如果特定的盘没有完成，所有的投注是无效的。
    • 如果特定盘有完成，但赛事并没有完成，那么所有投注在这个市场上的注单将被视为有效。

    总盘数：大/小

    • 预测赛事的总盘数将大于或小于赛事指定的大/小盘分数。
    • 如果赛事没有完成，那么所有的赌注将是无效的，除非赛事已有明确结果。

    首先达到3局

    • 预测哪一个选手将先赢得3 局。
    • 如果在比赛中没有任何一方赢取3局，所有的投注将被视为无效。

    在4局后得分

    • 预测在首4局已完成后的指定选手的正确得分。
    • 如果比赛的首4局没有完成，那么所有的投注都将是无效的。

    在4局后领先

    • 预测在比赛首4局已完成的情况下领先的选手。
    • 平局也可以供选择。
    • 如果比赛的首4局没有完成，所以的投注将被视为无效。

    波胆-X 盘

    • 预测在特定盘结束后的局数正确的比分。
    • 如果特定的盘没有完成，所有的投注将会视为无效。
    • 如果特定的盘完成，但是赛事没有完成，在这个市场所有的投注将会被确认。

    180分

    • 预测在特定时节内是否有180得分。
    • 这个市场可能基于赛事所有局中特定局或赛事其中一盘中的特定局。
    • 如果选手的名字没有包含在市场名单里面，这个结果将会基于任何一名选手的180分记录。
    • 如果球员名字在市场名称内，赛果将以该玩家在比赛180分的记录。
    • 如果特定局没有完成，所有这个市场的投注将被视为无效。

    180分－大/小盘

    • 预测赛事的总180分记录将大于或小于赛事指定的大/小180分记录。
    • 如果选手的名字没有包含在市场名单里面，结果将会基于两名选手的180分记录为准。
    • 如果球员名称在市场名称内，赛果将基于该玩家在比赛180总分的记
    • 如果比赛没有完成，所有的投注将会被视为无效，除非赛事已有明确结果。

    180分-让球

    • 预测赛事盘口在每个选手指定的让球数上的180分结果。
    • 结算将基于赛事结束后每个选手的180分记录总数为准。
    • 如果比赛没有完成，所有的投注将会被视为无效，除非赛事已有明确结果。

    获得最多180分

    • 预测在赛事中哪一位选手获得最多180分记录。
    • 平局也包含在选项里面。
    • 如果比赛没有完成，所有的投注将会被视为无效，除非赛事已有明确结果。

    决胜镖盘口

    • 预测哪一位选手将在赛事最后一局以决胜镖取胜。
    • 如果赛事没有完成，所 有的投注将被视为无效。

    X 决胜镖总数：大/小 X

    • 预测赛事的特定局的决胜镖总数大于或小于赛事指定的大/小决胜镖总数。
    • 如果赛事是以局为基准的，那么市场的提供将会以特定的局为基准。
    • 如果赛事是以盘为基准的，那市场的提供将会以这一盘的特定局为基准。
    • 如果特定局没有完成，所有的投注将被视为无效。

    决胜镖总数：大/小（ X－X）

    • 预测赛事中的特定局的决胜镖总数是在数值范围之间或决胜镖总数是高于或低于该数值范围。
    • 如果赛事是以局为基准的， 那么市场的提供将会以特定局为基准。
    • 如果赛事是以盘为基准的，那么市场的提供将会以这一局的特定局为基准。
    • 如果特定局没有完成，所有的投注将会被视为无效。

    最高决胜镖：大/小 X

    • 预测赛事的最高决胜镖是大于或小于指定的大/小。
      • 这个最高决胜镖可能会是170。
    • 如果一个选手的名字不在市场名单里面，结果将是基于双方选手的决胜镖记录结算。
    • 如果选手的名字包含在市场名单里面，这个结果将会基于特定选手的最高决胜镖。
    • 如果赛事没有完成，那么所有的赌注将是无效的，除非赛事已有明确结果。

    最高决胜镖：大/小 （X－X）

    • 预测赛事中的最高决胜镖是在数值范围之间或最高决胜镖是高于或低于该数值范围。这个最高决胜镖可能会是170。
    • 如果一个选手的名字不在市场名单里面，结果将是基于双方选手的决胜镖记录结算。
    • 如果选手的名字是包含在市场名单里面，结果将会以特定选手的最高决胜镖结算。
    • 如果赛事没有完成，那么所有的赌注将是无效的，除非赛事已有明确结果。

    决胜镖颜色

    • 预测在赛事特定期间的决胜镖颜色（例如：最后一镖）。
    • 如果比赛是以“局”为基础的，那么这个市场的提供将会以特定的局为基准。
    • 如果比赛是以“盘”为基础的，那么这个市场的提供将会以这一盘的特定局为基准。
    • 所提供的选择永远是“红色”或“绿色”。
    • 如果特定局没有完成，所有的投注将会被视为无效。
